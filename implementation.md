# Implementation Guidelines

## Structure
- Homepage implemented in Astro
- Optional component separation if useful
- Content stored as data where possible

## Styling
- Use CSS variables for colors and spacing
- Keep styles maintainable and readable

## Accessibility
- Semantic HTML
- Proper heading structure
- Accessible navigation and buttons

## Performance
- Minimize JavaScript
- Optimize assets
- Avoid unnecessary dependencies

## Maintainability
- Clear file organization
- Simple patterns over complex abstractions
