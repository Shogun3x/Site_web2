# Project Specs — AI & Automation Services Website

## Objective
Create a clean, high-end, futuristic service website for local SMEs in Québec City (Québec, Canada).
The site is a marketing / brochure website whose primary goal is lead generation.

Primary conversion goals:
- Discovery calls
- Direct contact by email

## Target Audience
Small and medium-sized businesses (PMEs), decision-makers, operations managers, and owners.
Audience is non-technical to semi-technical.

## Value Proposition
Automation and AI integration to reduce manual work, improve reliability, and optimize internal workflows.
Strong focus on pragmatic, measurable ROI.

## Core Services
- Workflow optimization
- Automation
- AI integration
- Consulting and custom development

## Tone & Brand
- Premium
- Technical
- Trustworthy
- Pragmatic (no hype)

## Site Type
- Single-page landing (homepage)
- Optional secondary pages (privacy, legal)

## Tech Constraints
- Built with Astro
- Static site (no backend required)
- Minimal JavaScript
- Fast load time

## Out of Scope
- Ecommerce
- User accounts
- Payments
- Advertising
- Blog (future optional)
