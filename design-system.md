# Design System

## Visual Direction
Futuristic, clean, and high-end.
Light grey surfaces with subtle depth.
Orange accents used sparingly to signal action and energy.

## Color Palette
- Background base: light grey gradient
- Primary text: near-black
- Secondary text: muted dark grey
- Borders: soft neutral grey
- Primary accent: orange
- Secondary accent: lighter orange
- Dark neutral: dark grey for secondary actions

## Typography
- Modern system font stack
- Clear hierarchy
- Generous line-height for readability

## Layout Principles
- Large margins and whitespace
- Clear visual hierarchy
- Consistent section spacing
- Rounded corners for cards and panels

## Components
- Navigation bar
- Hero section
- Content sections (cards, grids)
- Call-to-action blocks
- Footer

## Interaction Guidelines
- Subtle hover effects
- Clear focus states
- No aggressive animations

## Responsiveness
- Mobile-first
- Scales cleanly to large screens
